# 从C到C++的进阶之路：我的C++全景学习笔记与思考

> 前言：为什么要从C转向C++？
> 刚学完C语言时，我觉得自己已经掌握了控制计算机的钥匙（指针真奇妙！）。但是，每次处理字符串都要小心翼翼地考虑末尾的`\0`，每次实现动态数组都要写冗长的`malloc`和`free`。接触到C++后，我仿佛推开了一扇新世界的大门。这篇文章记录了我从C语言过渡到C++的完整心路历程，涵盖了基础语法、函数与结构体的进化，再到算法初探与数据结构入门。

## 第一阶段：C到C++的华丽转身 —— 基础语法的“降维打击”

学习C++的第一天，最直观的感受就是：代码变简洁了，而且变得更安全了。

## 1. 输入输出的革命：`cin` 与 `cout`
以前在C语言中，`scanf`和`printf`不仅要记住各种格式化占位符（`%d`, `%f`, `%s`），还经常因为忘记给变量加取地址符`&`而导致程序崩溃。
在C++中，流操作符完美解决了这个问题：

```cpp
#include <iostream>
using namespace std; // 引入标准命名空间

int main() {
    int age;
    double height;
    // C++会自动识别变量类型，告别 %d 和 %lf！
    cout << "请输入你的年龄和身高：" << endl; 
    cin >> age >> height;
    cout << "你今年 " << age << " 岁，身高 " << height << " 米。" << endl;
    return 0;
}

学习心得：cin/cout底层运用了C++的函数重载机制。虽然比scanf/printf略慢（可以通过关闭同步流优化），但开发效率极高！

2. 真假有别：bool 类型的引入
C语言中没有原生的布尔类型（要引入<stdbool.h>），通常用0和非0代替。C++原生支持了bool，取值只有true和false，代码语义瞬间清晰了很多。
