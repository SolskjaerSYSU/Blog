# 从C到C++的进阶之路：我的C++全景学习笔记与思考

> 前言：为什么要从C转向C++？
> 刚学完C语言时，我觉得自己已经掌握了控制计算机的钥匙。但是，每次处理字符串都要小心翼翼地考虑末尾的 `\0`，每次实现动态数组都要写冗长的 `malloc` 和 `free`。接触到 C++ 后，我仿佛推开了一扇新世界的大门。这篇文章记录了我从 C 语言过渡到 C++ 的完整心路历程，涵盖基础语法、函数与结构体的进化，再到算法初探与数据结构入门。

---

## 第一阶段：C到C++的华丽转身

学习 C++ 的第一天，最直观的感受就是：代码变简洁了，而且更安全了。

## 1. 输入输出的革命：`cin` 与 `cout`

以前在 C 语言中，`scanf` 和 `printf` 不仅要记住各种格式化占位符（`%d`、`%f`、`%s`），还经常因为忘记给变量加取地址符 `&` 而导致程序崩溃。  
在 C++ 中，流操作符完美解决了这个问题：

```cpp
#include <iostream>
using namespace std;

int main() {
    int age;
    double height;

    cout << "请输入你的年龄和身高：" << endl;
    cin >> age >> height;
    cout << "你今年 " << age << " 岁，身高 " << height << " 米。" << endl;

    return 0;
}
```

学习心得：`cin` / `cout` 底层运用了 C++ 的函数重载机制。虽然比 `scanf` / `printf` 略慢，但开发效率很高。

## 2. 真假有别：`bool` 类型的引入

C 语言中没有原生的布尔类型，通常用 `0` 和非 `0` 代替。  
C++ 原生支持 `bool`，取值只有 `true` 和 `false`，代码语义会清晰很多。
